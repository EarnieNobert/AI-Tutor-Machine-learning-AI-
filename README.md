# AI Learning Assistant (AI Tutor)

This project is an AI-powered tutoring assistant designed to support structured learning environments using large language models.

The application allows users to ask questions and receive AI-generated explanations to help reinforce learning concepts.

## Features

• AI tutoring responses using LLMs  
• Interactive question-and-answer learning  
• Progress tracking for user learning  

## Technologies Used

Python  
OpenAI API  
LangChain  
Jupyter Notebook  

## Project Files

app.py – main application logic  
sample.ipynb – experimentation and testing  
progress.json – stores learning progress  
requirements.txt – required Python packages
